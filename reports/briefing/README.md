# reports/briefing/

브리핑 시스템 v3.4 ↔ 종목분석 v2.4 통합 파이프라인의 **최종 산출물 저장소**.

## 역할

사용자가 매일 아침 읽는 **일일 브리핑** 1편이 저장된다.
`briefing-synthesizer` 가 `analysis/briefing/` 의 3개 분석 결과를 통합하여
이 폴더에 단일 파일로 작성한다.

## 저장 파일

```
reports/briefing/daily_briefing_{YYYYMMDD}.md   ← briefing-synthesizer
```

- 1일 1편 (KST 기준)
- 다른 에이전트는 이 폴더에 쓰지 않는다

## 접근 권한 매트릭스

| 에이전트 | 읽기 | 쓰기 |
|---|---|---|
| briefing-synthesizer | ✅ | ✅ |
| 그 외 모든 에이전트 | ❌ (사용자만 열람) | ❌ |

## 생명주기

- **생성:** 매 영업일, `/일일브리핑` 명령어 또는 상위 오케스트레이터 호출
- **유지:** 시계열 누적 (날짜별 파일 분리)
- **Git:** 커밋 대상. 단, Phase 5 통합 테스트 완료 후에만 push
- **공개 범위:** 사용자 + 본인 (외부 공유 금지 — 매수·매도 추천 아님 면책)

## 형식 사양

브리핑은 반드시 한국어로 작성되며, 다음 7개 섹션을 포함한다:

0. Executive Summary (3줄)
1. 오늘의 핵심 (Top 3 Headlines)
2. 시장 (market-analyst 인용)
3. 매크로 (macro-analyst 인용)
4. 거물 동향 (guru-analyst 인용, **13F 시차 경고 필수**)
5. Top 3 액션 아이템 (관찰 / 점검 / 리서치만 — 매수·매도 절대 금지)
6. 다음 영업일 트리거 일정
7. 한계와 주의

자세한 사양은 `.claude/agents/briefing-synthesizer.md` 의 **산출물 구조** 절을 참조.

## 절대 금지 사항 (산출물 검증)

브리핑 파일은 발행 전 자가 검증을 거친다:

- ❌ 매수·매도·익절·손절·비중조정 표현 0건
- ❌ 목표주가·손절가·진입가 수치 0건
- ❌ 13F 시차 고지 누락
- ❌ Top 3 액션 항목 ≠ 정확히 3개
- ❌ 영어 본문
