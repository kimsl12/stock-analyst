| trail_trigger_pct | 10% | 기본값 적용 |
| rr_ratio | 2.0 | 기본값 적용 |

#### STEP 1 — 초기 손절가 계산

```

atr_stop   = $361.26 - ($15.20 x 2) = $330.86

initial_stop = MAX($332.36, $330.86) = $332.36
used_method = "고정비율" (더 타이트한 쪽)
```

#### STEP 2 — 트레일링 전환 판단

```
trail_threshold = $361.26 x (1 + 0.10) = $397.39
is_trailing = (current_high >= $397.39) ? -> 현재 미전환 (FIXED 모드)
```

#### STEP 3 — 트레일링 손절가 (전환 후 적용)

```
전환 후: trailing_stop = current_high - ($15.20 x 2)
final_stop = MAX(trailing_stop, prev_stop)
-> 현재는 FIXED 모드이므로 적용 대기
```

#### STEP 4 — 목표가 (손익비 기반)

