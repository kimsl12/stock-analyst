---
argument-hint: [종목명]
description: 분석 결과 파일을 읽어 HTML 리포트를 생성합니다. 다른 세션의 결과물도 사용 가능. 예) /리포트 삼성전자
agent: report-generator
---

$ARGUMENTS 종목의 분석 결과를 HTML 리포트로 생성해줘.

데이터 소스 우선순위:
1. reports/ 폴더에서 해당 종목의 .md 파일을 검색
2. 파일이 있으면 → 파일 내용을 읽어서 HTML 리포트 생성
3. 파일이 없으면 → 현재 대화 컨텍스트에서 분석 데이터 사용
4. 둘 다 없으면 → "분석 결과를 찾을 수 없습니다. /종목분석을 먼저 실행해주세요." 안내

파일 검색 패턴:
  - reports/ 폴더에서 종목명 또는 종목코드로 검색
  - 가장 최근 날짜 파일 채택

HTML 생성 후:
  - reports/ 폴더에 HTML 저장
  - git add + commit + push
