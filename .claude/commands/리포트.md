$ARGUMENTS 종목의 분석 결과를 HTML 리포트로 생성해줘.

현재 대화에 있는 분석 데이터를 사용하여:
1. 모바일 퍼스트 반응형 HTML 단일 파일 생성
2. 결론 우선 (Executive Summary 상단 고정)
3. 각 섹션 접이식 카드 레이아웃
4. 손절/목표가 카드 포함
5. SVG 차트 (스코어카드 레이더, 실적 바차트, 리스크 히트맵)
6. 파일을 다운로드 가능하게 제공
