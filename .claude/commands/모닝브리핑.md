$ARGUMENTS 인자로 **모닝 브리핑 (MODULE A)** 를 실행해줘.

## 명령 정보

- **모듈:** A (briefing_module_AB.md MODULE A)
- **산출물:** `reports/briefing/morning_{YYYYMMDD}.html`
- **시점:** 매일 아침 KST 07~10시 (어젯밤 미국장 마감 직후)

## 인자 해석

- 인자 없음 → 대상일 = **오늘 KST**
- `YYYYMMDD` → 그 날짜를 대상일로 사용
- `--skip-collect` → market-data-collector 호출 생략 (이미 오늘자 KB 최신인 경우만)
- `--html` → HTML 동시 생성 (기본 활성)

## briefing-lead 에 전달할 컨텍스트

```
mode: morning
target_date: {YYYYMMDD}
region_focus: us
include_13f: false
sections (briefing_module_AB.md MODULE A 기준):
  - A-1 미국 시장 마감 핵심 (지수/VIX/금리/환율/원자재/크립토)
  - A-2 어젯밤 주요 뉴스 3~5건 ([소스] + 영향도 + 영향 방향)
  - A-3 오늘 아시아 시장 전망
  - A-4 오늘의 체크포인트 (주시 vs 무시)
  - A-5 거물 투자자 동향 스냅샷 (8인 워치리스트 — Cathie Wood 포함)
  - A-6 모델 포트폴리오 4종 방향 체크 (안전/중립/공격/배당)
  - A-7 글로벌 매크로 핵심 시사점 (MODULE G 연계 1~2건)
required_extras:
  - debate-card 1건 이상
  - contrarian-card 1건 이상
  - 13F 시차 경고 (A-5 거물 인용)
  - 4종 포트폴리오 방향 (전부)
template: morning
```

briefing-lead 의 워크플로 16단계를 그대로 진행. 산출물 경로만 `morning_{YYYYMMDD}` 로 교체.
