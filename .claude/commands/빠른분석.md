$ARGUMENTS 종목을 웹 검색으로 빠르게 분석해줘. 서브에이전트 호출 없이 직접 분석.

다음 항목을 간략히 정리:
1. 현재가 / 시가총액 / 52주 범위
2. PER / PBR / ROE (최신 기준)
3. 최근 실적 (직전 분기)
4. 컨센서스 목표주가 (평균)
5. 핵심 투자포인트 3줄
6. 핵심 리스크 3줄
7. 한줄 투자의견

추가로 ATR 기반 손절/목표가를 계산해줘:
- 최근 20거래일 일봉 데이터를 웹 검색으로 추정 (정확한 데이터 없으면 52주 변동폭 기반 ATR 추정)
- stop-loss-rules.md의 공식 적용:
  fixed_stop = 현재가 × 0.92 (8% 고정손절)
  atr_stop = 현재가 - (ATR14 × 2)
  initial_stop = MAX(fixed_stop, atr_stop)
  target = 현재가 + (현재가 - initial_stop) × 2

결과를 아래 형식으로 표시:

```
🔴 손절가  ₩XXX,XXX  (-X.X%)  [고정비율/ATR 채택]
🟢 목표가  ₩XXX,XXX  (+X.X%)  R:R = 1:2
```

ATR 정확한 값을 못 구하면 "ATR 추정치 사용"으로 표기하고 고정비율 8% 기준으로만 계산.

5분 이내 완료를 목표로 간결하게 작성.
