$ARGUMENTS 종목을 웹 검색으로 빠르게 분석해줘. 서브에이전트 호출 없이 직접 분석.

## 응답 순서 (반드시 이 순서를 지킨다)

### STEP 1: 텍스트 리포트를 먼저 사용자에게 보여준다

다음 항목을 간략히 정리:
1. 현재가 / 시가총액 / 52주 범위
2. PER / PBR / ROE (최신 기준)
3. 최근 실적 (직전 분기)
4. 컨센서스 목표주가 (평균)
5. 핵심 투자포인트 3줄
6. 핵심 리스크 3줄
7. ATR 손절/목표가:
   fixed_stop = 현재가 × 0.92 (8% 고정손절)
   atr_stop = 현재가 - (ATR14 × 2)
   initial_stop = MAX(fixed_stop, atr_stop)
   target = 현재가 + (현재가 - initial_stop) × 2
   → 🔴 손절가 ₩XXX (-X%) 🟢 목표가 ₩XXX (+X%)
8. 한줄 투자의견

ATR 정확한 값을 못 구하면 고정비율 8% 기준으로만 계산하고 "ATR 추정치 사용" 표기.

### STEP 2: 텍스트 응답 완료 후, 파일 저장 + Git 푸시를 백그라운드로 실행

사용자에게 텍스트를 모두 보여준 다음에 아래를 실행한다:

```bash
# 1. reports 폴더에 텍스트 리포트 저장
mkdir -p reports
cat > "reports/{종목코드}_{종목명}_빠른분석_{YYYYMMDD}.md" << 'EOF'
(위에서 보여준 텍스트 리포트 내용)
EOF

# 2. Git 커밋 + 푸시
git add reports/
git commit -m "빠른분석: {종목명} - {YYYY-MM-DD}"
git push origin main
```

- 저장/푸시 성공 시: "📁 reports/에 저장 완료" 한줄 안내
- 저장/푸시 실패 시: 무시 (사용자에게 오류 표시하지 않음, 텍스트 응답은 이미 전달됨)

5분 이내 완료를 목표로 간결하게 작성.
