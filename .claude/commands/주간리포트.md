$ARGUMENTS 인자로 **주간 리포트 (MODULE C)** 를 실행해줘.

## 명령 정보

- **모듈:** C (briefing_module_CDE.md MODULE C)
- **산출물:** `reports/briefing/weekly_{YYYYMMDD}.html`
- **시점:** 매주 금요일 KST 18~22시
- **데이터 범위:** 최근 5영업일 (월~금)
- **특징:** Phase 0-D 성과 추적 단계 추가

## 인자 해석

- 인자 없음 → 오늘 KST (이번 주 금요일)
- `YYYYMMDD` → 해당 주차 마감일
- `--skip-collect` → market-data-collector 생략
- `--html` → HTML 동시 생성 (기본)

## briefing-lead 에 전달할 컨텍스트

```
mode: weekly
target_date: {YYYYMMDD}
sections (briefing_module_CDE.md MODULE C):
  - C-1 이번 주 시장 총평 (스파크라인)
  - C-2 경제 환경 심층 + 다음 주 발표 일정
  - C-3 지정학·국제 정세 영향 (2차·3차 효과)
  - C-3.5 기술·에너지 주간 인사이트 (단계 판정)
  - C-4 큰손 동향 + 거물 8인 주간 종합 (컨버전스 시그널)
  - C-5 주목 종목·ETF 스크리닝 (S~D 등급)
  - C-6 다음 주~한 달 핵심 일정
  - C-7 투자 전략 제안 (3 시나리오)
  - C-8 리스크 레이더 (놓치는 위험 3가지)
  - C-9 ★ 과거 제안 성과 추적 ★ (Phase 0-D)
  - C-10 주간 모델 포트폴리오 4종 종합 방향
required_extras:
  - debate-card 1건 이상
  - contrarian-card 1건 이상
  - 13F 시차 경고 (C-4)
  - 4종 포트폴리오 방향
  - global-macro-analyst 호출 필수 (mode=weekly, C-3 + C-3.5)
  - correlation-monitor 호출 필수 (mode=weekly_summary)
phase_0_d_performance_tracking:
  - knowledge-db/performance/2026_recommendations.md 읽기 (지난 1주~1개월)
  - market-data-collector 가 수집한 실제 가격으로 적중률 판정
  - knowledge-db/performance/2026_hit_rate.md append
template: weekly
```

briefing-lead 워크플로 + Phase 0-D 추가. 산출물 `weekly_{YYYYMMDD}`.
