$ARGUMENTS 인자로 **이브닝 브리핑 (MODULE B)** 를 실행해줘.

## 명령 정보

- **모듈:** B (briefing_module_AB.md MODULE B)
- **산출물:** `reports/briefing/evening_{YYYYMMDD}.html`
- **시점:** 매일 저녁 KST 18~22시 (아시아장 마감 후)

## 인자 해석

- 인자 없음 → 오늘 KST
- `YYYYMMDD` → 해당 날짜
- `--skip-collect` → market-data-collector 생략
- `--html` → HTML 동시 생성 (기본)

## briefing-lead 에 전달할 컨텍스트

```
mode: evening
target_date: {YYYYMMDD}
region_focus: both
include_13f: false
sections (briefing_module_AB.md MODULE B):
  - B-1 글로벌 이슈 탑 5 (각 2차 효과 1줄)
  - B-2 아시아 시장 마감 (KOSPI/KOSDAQ/N225/SSE/HSI/SENSEX)
  - B-3 투자 방향 신호판 (Signal Dashboard 6종)
  - B-4 경제 서프라이즈 인덱스 ★ (correlation-monitor 산출)
  - B-5 자산 간 상관관계 변화 모니터 ★ (correlation-monitor 산출, 6쌍 Z-score)
  - B-6 신규 투자 아이디어 0~2개 (주식/ETF/토큰)
  - B-7 거물 투자자 동향 심화 (8인 의견 대비표)
  - B-8 모델 포트폴리오 4종 방향 (아침 대비 변화 컬럼 포함)
  - B-9 글로벌 매크로 핵심 시사점 (MODULE G 연계)
required_extras:
  - debate-card 1건 이상
  - contrarian-card 1건 이상
  - 13F 시차 경고 (B-7)
  - 4종 포트폴리오 방향
  - correlation-monitor 호출 필수 (mode=full)
template: evening
```

briefing-lead 의 워크플로 16단계 그대로. 산출물 경로 `evening_{YYYYMMDD}`.
