$ARGUMENTS 인자로 **모델 포트폴리오 (MODULE F)** 를 실행해줘.

## 명령 정보

- **모듈:** F (briefing_module_F.md)
- **산출물:** `reports/briefing/model_portfolio_{YYYYMMDD}.html`
- **목적:** 4종 현재 구성 (교육·참고용) + 구체적 종목·ETF 티커 (웹 서치 후)
- **차별점 vs /리밸런싱:** /리밸런싱은 변화 방향, /모델포트폴리오는 현재 스냅샷 + 종목 지정

## 인자 해석

- 인자 없음 → 오늘 KST
- `YYYYMMDD` → 해당 일자
- `--html` → HTML 생성 (기본)

## briefing-lead 에 전달할 컨텍스트

```
mode: model_portfolio
target_date: {YYYYMMDD}
sections (briefing_module_F.md):
  - F-1 현재 시장 환경 진단 (포트폴리오 구성의 전제)
  - F-2 🛡️ 안전형 (자산군별 비중표 + 구체 종목/ETF + 리스크 프로파일)
  - F-3 ⚖️ 중립형 (동일 구조)
  - F-4 🔥 공격형 (동일 구조)
  - F-5 💰 배당형 (동일 구조 + 배당 시뮬레이션)
  - F-6 4종 한눈에 비교표
  - F-7 ⚠️ 모델 포트폴리오 주의사항 (필수 표시)
required_extras:
  - 4종 전부 (강조)

  - 거물 8인 매핑 (안전형: Browne+Dalio, 중립형: Swensen+Buffett 등)
  - debate-card 1건 (예: 채권 비중 vs 크립토 비중 논쟁)

  - 함정 배당 무경고 제시 금지 (#19)
required_actions:
  - market-data-collector 로 F-1 환경 진단 데이터 우선 수집
  - briefing-lead 가 직접 web search 로 최신 종목·ETF 정보 확보
template: model_portfolio
```
