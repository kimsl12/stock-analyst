$ARGUMENTS 인자로 **크립토 브리핑 (MODULE E)** 를 실행해줘.

## 명령 정보

- **모듈:** E (briefing_module_CDE.md MODULE E)
- **산출물:** `reports/briefing/crypto_{YYYYMMDD}.html`
- **데이터 소스:** CoinGecko (1차) + Glassnode + DeFiLlama + Messari

## 인자 해석

- 인자 없음 → 오늘 KST
- `YYYYMMDD` → 해당 날짜
- `--skip-collect` → 생략
- `--html` → HTML 생성 (기본)

## briefing-lead 에 전달할 컨텍스트

```
mode: crypto
target_date: {YYYYMMDD}
sections (briefing_module_CDE.md MODULE E):
  - E-1 크립토 시장 대시보드 (BTC/ETH/SOL + 시총 + 도미넌스 + Fear & Greed)
  - E-2 온체인 핵심 지표 (거래소 잔고, 장기보유자, TVL)
  - E-3 크립토-전통자산 상관관계 (BTC vs NASDAQ/Gold/USD — correlation-monitor 산출)
  - E-4 규제 & 제도권 편입 동향
  - E-5 주목 토큰/프로토콜 0~3개
  - E-6 대체자산 스냅샷 (해당 시 — Gold, REITs 등)
required_extras:
  - debate-card 1건 (예: BTC ETF 자금 흐름, L2 vs L1, AI×크립토)
  - contrarian-card 1건
  - correlation-monitor 호출 (mode=crypto)
forbidden:
  - 13F 시차 경고 생략 가능 (거물 인용 없음)
  - 4종 포트폴리오 방향 생략 가능
template: crypto
```
