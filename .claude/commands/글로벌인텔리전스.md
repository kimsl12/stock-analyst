$ARGUMENTS 인자로 **글로벌 인텔리전스 (MODULE G)** 를 실행해줘.

## 명령 정보

- **모듈:** G (briefing_module_G.md)
- **산출물:** `reports/briefing/global_intelligence_{YYYYMMDD}.html`
- **시간축:** 단기(1~3개월) / 중기(3~12개월) / 장기(1~5년)
- **핵심 분석가:** global-macro-analyst (mode=full) 단독 호출 후 briefing-lead 통합

## 인자 해석

- 인자 없음 → 오늘 KST
- `YYYYMMDD` → 해당 일자
- `--skip-collect` → market-data-collector 생략 (--skip-collect 권장 — KB macro/ 가 메인)
- `--html` → HTML 생성 (기본)

## briefing-lead 에 전달할 컨텍스트

```
mode: global_intelligence
target_date: {YYYYMMDD}
sections (briefing_module_G.md):
  - G-1 지정학 파워맵 + 공급망 재편 (5축 + 5산업)
  - G-2 정치 사이클 + 정책 → 산업 매핑 + 캘린더
  - G-3 첨단 기술 트래커 (단계 판정 🔬→🧪→🏭→🌍)
  - G-4 에너지 전환 + 핵심 광물 + 기후 + AI 데이터센터 전력
  - G-5 ★ 4축 교차 메가트렌드 ★ (2축=일반, 3축=강화, 4축=메가, 숨은 테마★)
  - G-6 2차·3차 효과 + 과거 유사 사례
  - G-7 글로벌 자본 흐름 + 달러 사이클
  - G-8 ★ 시나리오 플래닝 ★ (3~6개월 갈림길 + 4종 포트폴리오 대응)
  - G-9 ⚠️ MODULE G 주의사항 (필수)
required_extras:
  - debate-card 1건 (4축 중 어느 축이 가장 강한 동인인가)
  - contrarian-card 1건 (시장이 가격에 반영 안 한 4축 교차점)
  - 시나리오 트리 시각화 (G-8)
  - 기술 단계 판정 (G-3) — 누락 금지
required_actions:
  - global-macro-analyst 호출 (mode=full) — 본 명령의 핵심
  - 시나리오 분기점은 knowledge-db/performance/2026_scenario_tracking.md 에 append
forbidden:
  - 1차 효과만 분석하고 멈추기 (#19)
  - 지정학·정치를 시장 외부 변수로 취급 (#20)
  - 기술을 단계 판정 없이 나열 (#21)
  - 13F 시차 경고 생략 가능
  - 4종 포트폴리오 방향 생략 가능 (G-8 시나리오 대응표가 대신 — 시나리오별 4종)
template: global_intelligence
```

briefing-lead 워크플로. global-macro-analyst (Opus) 가 큰 산출물을 작성하고 briefing-lead 가 종합.
