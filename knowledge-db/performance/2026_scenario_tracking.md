> **목적:** G-8 시나리오 플래닝에서 제시된 갈림길의 확률 변화를 시계열 추적.
> **갱신 주체:** `/주간리포트` 또는 `/글로벌인텔리전스` 실행 시 briefing-lead 가 갱신.

```
#### 확률 추이 (행 1개 = 1회 갱신)

| 갱신일 | A 확률 | B 확률 | C 확률 | 변화 이유 | 출처 |

| 항목 | 값 |

| 교훈 | (1~2줄) |
```

## 갱신 규칙 (briefing-lead 워크플로)

1. **신규 시나리오 추가**: `/글로벌인텔리전스` 실행 시 G-8-1 에서 새 갈림길이 식별되면 본 파일에 시나리오 섹션 추가.
2. **확률 재추정**: 매 `/주간리포트` 실행 시 활성 시나리오 전부에 대해 새 확률 행 append.
   변화가 ±5%p 미만이면 "변화 없음" 1줄로 대체 가능.
3. **종결 처리**: 시간축 종료일이 도래하면 즉시 종결 처리 + 아카이브로 이동.
   결과 데이터는 `knowledge-db/market/2026_daily_prices.md` 에서 확인.
4. **확률 합계 검증**: A+B+(C) 합계는 항상 100%. 99% 또는 101% 허용 (반올림).
